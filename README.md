# Phish_Guard AI - Advanced Phishing Detection

Multi-modal detector: URL + Email + SMS (Smishing) + Typo Squatting

**Features**
- 35+ real-time URL features
- Keras LSTM for Email & SMS
- Typo Squatting detection
- Hybrid analysis
- PDF report download

**Accuracy**: URL ~97% | Text ~96%

**How to run locally**  
```bash
pip install -r requirements.txt
streamlit run app.py
